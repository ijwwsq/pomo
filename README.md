# pomo

a tiny, quiet pomodoro timer for macos. keeps you focused, keeps it minimal, keeps it moving. just a small fun thing.

usage
```sh
./pomo                  # default 50/10 × 4 cycles
./pomo 25 5 6           # positional: work/break/cycles
./pomo -w 30 -b 5 -c 3  # flags
```
terminal shows a soft countdown, notifications whisper when it’s time.

**install**
```sh
chmod +x pomo
mv pomo /usr/local/bin
```
philosophy
- tiny
- quiet
- sharp
- does one thing, does it well
